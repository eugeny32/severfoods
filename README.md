# 🍽️ CANTEEN ACCESS SYSTEM v2.0
## Многоточечная система контроля питания — СЕВЕР

---

## 📋 Описание

Полноценная веб-система пропускного контроля для сети корпоративных столовых:
- **Неограниченное количество точек** питания по всей стране
- Сканирование QR-кодов (USB-сканер, камера телефона)
- Автоматическое определение типа питания по расписанию
- Роли: Супер-администратор / Администратор / Оператор
- Детализированные отчёты с экспортом CSV
- PWA — устанавливается на телефон как приложение

## 👥 Роли пользователей

| Роль | Возможности |
|------|-------------|
| **Оператор** | Сканирование QR, ручной пропуск, просмотр сотрудников |
| **Администратор** | Всё выше + управление сотрудниками, отчёты, расписание своей точки |
| **Супер-администратор** | Всё + управление точками питания, все отчёты, удаление |

### Демо-доступ
- Администратор: логин `admin`, пароль `admin123`
- Оператор: QR-код `SUPER_ADMIN_QR` (указать любую точку)

---

## 🏗️ Добавление новых точек питания

1. Войдите как **супер-администратор**
2. Откройте **Вкладка "Точки" → "Управление точками"**
3. Добавьте точку: название, уникальный код, город
4. Настройте **расписание** для каждой точки (⏰ Расписание)
5. Назначьте **операторов** — добавьте сотрудников с ролью "Оператор"

## 🔒 Безопасность

- Смените пароль `admin123` на боевом сервере
- Включите HTTPS (`secure: true` в config.php → cookie params)
- Регулярно очищайте `logs/` и `temp_qr/`
- Ограничьте доступ к `config.php` через `.htaccess`
