# SeverFoods Offline Changelog

## v1.3.0 (2026-06-13)

### QR-очередь
- Замена `scanCooldown` на FIFO-очередь: сканы при обработке идут в очередь, а не игнорируются
- Дубли одного кода в очереди пропускаются (защита от быстрой камеры)
- Счётчик очереди отображается в попапе результата

### Всплывающее окно — полный редизайн
- Большой баннер снизу на всю ширину контентной зоны
- Ярко-зелёный фон = OK, оранжевый = повтор, ярко-красный = отказ/блокировка
- Крупный шрифт: 56px иконка, 30px имя, 20px сообщение
- Красный держится 8 секунд, остальные 3 секунды
- Приоритет: ошибка не перебивается успешными/повторными сканами

### Звук
- Отказ: 5-секундная тревога (чередующиеся тоны 880/660 Гц), не блокирует очередь
- OK: двухнотный восходящий сигнал (sine)
- Повтор: двухнотный нейтральный сигнал (triangle)
- Новая ошибка перезапускает тревогу с начала

### Увеличенный интерфейс для слабовидящих
- Цифры статистики: 32px (было 22px)
- QR-поле ввода: 17px, увеличенные отступы
- Кнопки навигации: 14px, больше padding
- Строки журнала сканирования: 14px
- Заголовки страниц: 22px

### Сборка
- Версия: 1.0.0 → 1.3.0
- Добавлен scripts/make-update.js для создания ZIP-пакета обновления
